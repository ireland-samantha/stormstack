=========
Tutorials
=========

Learn to build games on StormStack through hands-on tutorials.

These tutorials build on each other, so we recommend completing them in order.

Prerequisites
-------------

Before starting these tutorials, complete the :doc:`/getting-started/quickstart`.

Tutorial Path
-------------

.. list-table::
   :header-rows: 1
   :widths: 30 20 50

   * - Tutorial
     - Time
     - What You'll Learn
   * - :doc:`hello-stormstack`
     - 15 min
     - Explore containers, matches, commands, and snapshots
   * - :doc:`first-module`
     - 30 min
     - Create a HealthModule with HP tracking

.. toctree::
   :maxdepth: 2
   :caption: Tutorials:

   hello-stormstack

Learning Objectives
-------------------

By the end of these tutorials, you'll understand:

* How Execution Containers isolate game instances
* How the tick cycle processes commands and updates state
* How to create modules with components, systems, and commands
* How to deploy and test your modules

After the Tutorials
-------------------

Once you've completed the tutorials:

* :doc:`/concepts/index` - Deep dive into core concepts
* :doc:`/thunder/engine/index` - Engine internals and advanced features
* :doc:`/api/index` - REST and WebSocket API reference
