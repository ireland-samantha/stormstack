Authentication Endpoints

Login

POST /api/auth/login

Authenticate with username and password to receive a JWT token.

Request:

{
  "username": "admin",
  "password": "secret123"
}

Response (200 OK):

{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "expiresIn": 3600,
  "tokenType": "Bearer"
}
Status Codes:

  • 200 OK – Authentication successful

  • 401 Unauthorized – Invalid credentials

Token Refresh

POST /api/auth/refresh

Refresh an expiring JWT token.

Headers:

  • Authorization: Bearer <current-token>

Response (200 OK):

{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "expiresIn": 3600
}

Container Endpoints

List Containers

GET /api/containers

List all execution containers.

Response (200 OK):

{
  "containers": [
    {
      "id": 1,
      "name": "production",
      "status": "RUNNING",
      "matchCount": 5,
      "tickRate": 60,
      "currentTick": 15420
    }
  ]
}

Create Container

POST /api/containers

Create a new execution container.

Request:

{
  "name": "staging",
  "tickRate": 30,
  "maxEntities": 100000
}

Response (201 Created):

{
  "id": 2,
  "name": "staging",
  "status": "CREATED"
}
Status Codes:

  • 201 Created – Container created

  • 400 Bad Request – Invalid configuration

  • 403 Forbidden – Insufficient permissions

Get Container

GET /api/containers/(int: containerId)

Get details for a specific container.

Parameters:

  • containerId (integer) – Container ID

Response (200 OK):

{
  "id": 1,
  "name": "production",
  "status": "RUNNING",
  "tickRate": 60,
  "currentTick": 15420,
  "modules": ["EntityModule", "RigidBodyModule"],
  "matches": [
    {"id": 1, "playerCount": 4, "entityCount": 150}
  ]
}

Delete Container

DELETE /api/containers/(int: containerId)

Delete a container. Container must be stopped first.

Status Codes:

  • 204 No Content – Container deleted

  • 400 Bad Request – Container not stopped

  • 404 Not Found – Container not found

Match Endpoints

List Matches

GET /api/containers/(int: containerId)/matches

List all matches in a container.

Response (200 OK):

{
  "matches": [
    {
      "id": 1,
      "containerId": 1,
      "playerCount": 4,
      "entityCount": 150,
      "status": "RUNNING",
      "createdAt": "2026-02-03T10:00:00Z"
    }
  ]
}

Create Match

POST /api/containers/(int: containerId)/matches

Create a new match in the container.

Request:

{
  "metadata": {
    "gameMode": "deathmatch",
    "maxPlayers": 8
  }
}

Response (201 Created):

{
  "id": 2,
  "containerId": 1,
  "status": "CREATED"
}

Command Endpoints

Queue Commands

POST /api/containers/(int: containerId)/commands

Queue commands for execution in the next tick.

Request:

{
  "matchId": 1,
  "commands": [
    {
      "name": "SpawnEntity",
      "params": {
        "x": 100.0,
        "y": 50.0,
        "type": "player"
      }
    }
  ]
}

Response (202 Accepted):

{
  "queued": 1,
  "nextTick": 15421
}
Status Codes:

  • 202 Accepted – Commands queued

  • 400 Bad Request – Invalid command format

  • 404 Not Found – Match not found

Get Command Schema

GET /api/containers/(int: containerId)/commands/schema

Get the schema for all available commands in the container.

Response (200 OK):

{
  "commands": [
    {
      "name": "SpawnEntity",
      "module": "EntityModule",
      "parameters": [
        {"name": "x", "type": "float", "required": true},
        {"name": "y", "type": "float", "required": true},
        {"name": "type", "type": "string", "required": true}
      ]
    }
  ]
}

Simulation Endpoints

Advance Tick

POST /api/containers/(int: containerId)/ticks

Manually advance the simulation by one or more ticks.

Request:

{
  "count": 1
}

Response (200 OK):

{
  "previousTick": 15420,
  "currentTick": 15421,
  "commandsExecuted": 3
}

Start Auto-Advance

POST /api/containers/(int: containerId)/ticks/start

Start automatic tick advancement at the configured tick rate.

Response (200 OK):

{
  "status": "RUNNING",
  "tickRate": 60
}

Stop Auto-Advance

POST /api/containers/(int: containerId)/ticks/stop

Stop automatic tick advancement.

Response (200 OK):

{
  "status": "PAUSED",
  "currentTick": 15421
}

Snapshot Endpoints

Get Snapshot

GET /api/containers/(int: containerId)/snapshots/(int: matchId)

Get the current ECS snapshot for a match.

Query Parameters:

  • format - Response format: json (default) or binary

  • modules - Comma-separated list of modules to include

Response (200 OK):

{
  "matchId": 1,
  "tick": 15421,
  "modules": {
    "EntityModule": {
      "components": ["Position", "Velocity"],
      "entities": [
        {
          "id": 1,
          "Position": {"x": 100.0, "y": 50.0},
          "Velocity": {"dx": 1.0, "dy": 0.0}
        }
      ]
    }
  }
}

Module Endpoints

List Modules

GET /api/containers/(int: containerId)/modules

List installed modules in a container.

Response (200 OK):

{
  "modules": [
    {
      "id": "EntityModule",
      "version": "1.0.0",
      "components": ["Position", "Velocity", "Entity"],
      "commands": ["SpawnEntity", "DestroyEntity"],
      "systems": ["MovementSystem"]
    }
  ]
}

Install Module

POST /api/containers/(int: containerId)/modules

Install a module JAR file into the container.

Content-Type: multipart/form-data

Form Fields:

  • file - The module JAR file

Response (201 Created):

{
  "id": "CustomGameModule",
  "version": "1.0.0",
  "status": "INSTALLED"
}

Control Plane Endpoints

Cluster Status

GET /api/cluster/status

Get overall cluster health and status.

Response (200 OK):

{
  "totalNodes": 3,
  "healthyNodes": 3,
  "drainingNodes": 0,
  "totalCapacity": 300,
  "usedCapacity": 45,
  "saturation": 0.15
}

List Nodes

GET /api/nodes

List all registered nodes in the cluster.

Response (200 OK):

{
  "nodes": [
    {
      "id": "node-1",
      "url": "http://engine-1:8080",
      "status": "HEALTHY",
      "capacity": 100,
      "activeMatches": 15,
      "lastHeartbeat": "2026-02-03T10:30:00Z"
    }
  ]
}

Route Match

POST /api/matches/route

Route a new match to the optimal node based on capacity.

Request:

{
  "gameType": "battle-royale",
  "expectedPlayers": 100,
  "region": "us-west"
}

Response (200 OK):

{
  "nodeId": "node-2",
  "nodeUrl": "http://engine-2:8080",
  "containerId": 1,
  "matchId": 42
}

Error Codes

Code

Description

AUTHENTICATION_REQUIRED

No valid JWT token provided

PERMISSION_DENIED

User lacks required role/permission

CONTAINER_NOT_FOUND

Container ID does not exist

MATCH_NOT_FOUND

Match ID does not exist

CONTAINER_NOT_RUNNING

Container must be running for this operation

INVALID_COMMAND

Command name or parameters invalid

MODULE_CONFLICT

Module version conflict during install

CAPACITY_EXCEEDED

Cluster at capacity, cannot route match
